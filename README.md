# trip-route-planner

## Test the application

You can run in docker environment for easier set up.

You can check the application with this input json:

```json lines
{
    "trips": {
        "first": 0,
        "second": 'third',
        "third": 'sixth',
        "fourth": 'first',
        "fifth": 'second',
        "sixth": 0,
    }
}
```
Or you can run the Feature tests.
